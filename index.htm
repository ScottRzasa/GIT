Getting Started with Git in VS Code
Outline
Setting Up the Environment
Install Git and VS Code. Configure Git within VS Code (user name, email).
Creating and Cloning a GitHub Repository
Create a new repository on GitHub. Clone the repository using VS Code's integrated Git support.
Basic Git Operations in VS Code
Perform basic Git operations: status, add, commit, push, and pull through the VS Code GUI.
Branching with VS Code
Use VS Code to create and switch between branches. Understand the concept of branching for feature development or bug fixes.
Working with Remote Repositories
Add and manage remote repositories directly from VS Code. Push to and pull from these remotes using the VS Code interface.
Pull Requests and Code Reviews within VS Code
Create pull requests using GitHub and VS Code. Review and discuss pull requests.

Setting Up the Environment

Install Git and VS Code.

To install Git on to your computer serach "Git" into your preferred browser and click the link thats shown as follows.

Once on the main page find and click the button in the bottom right corner of screen that says "Download for 'Whatever system you have'".
Then choose and click the installer you want to use (Recommened 64-bit).
Navigate to you downloads and download the git file you chose. Open it once downloaded.
Once on the setup page the instilation for Git may begin.
